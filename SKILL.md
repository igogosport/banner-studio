# Banner Studio — 電商美編協作系統

三角色協作的電商促銷圖產生器。美編設定視覺模板，電商填入商品資料，行銷選擇平台尺寸，一鍵輸出所有格式的真實 PNG 圖檔。

## 系統架構

三角色分工：
  🛒 電商同仁  → 商品名稱、特價/原價、活動日期、平台、商品圖
  📣 行銷同仁  → 貼文文案、Hashtag、勾選輸出尺寸（FB/IG/電商/LINE）
  🎨 美編同仁  → 上傳套版底圖、設定文字顏色字型、拖曳調整區塊位置

輸出：單一 HTML 檔案（可直接部署為網站，不需後端）
匯出：Canvas API 繪圖 → 真實 PNG 下載（非截圖）

## 內建尺寸規格

Facebook：動態貼文 1200x630、限時動態 1080x1920、封面 820x312、方形 1080x1080
Instagram：方形 1080x1080、直式 1080x1350、限時動態 1080x1920、橫式 1080x608
電商平台：800x600、1024x768、橫幅 1200x300、方形 800x800、全寬 1920x600
LINE/其他：LINE圖文訊息 1040x700、YouTube縮圖 1280x720

## 文字區塊預設座標（百分比，可拖曳）

badge（標語）：x=5, y=5
title（商品名稱）：x=5, y=22
price（特價金額）：x=5, y=68
date（活動日期）：x=60, y=78
product（商品圖片區）：x=60, y=4

## Canvas 輸出技術重點

- 使用瀏覽器原生 Canvas 2D API，不依賴 html2canvas
- 漸層背景：createLinearGradient 根據角度計算向量
- 商品圖片：cover fit + 圓角 clip + drop shadow
- 字型：等待 document.fonts.ready 確保字型載入完成再繪製
- 多檔下載：每張間隔 350ms 避免瀏覽器封鎖
- 輸出檔名格式：{商品名}_{尺寸標籤}_{寬}x{高}.png

## 美編可調整項目

- 底圖：上傳自訂 PNG/JPG 或使用內建漸層
- 字型：Anton / Bebas Neue / Bangers / Noto Sans TC
- 顏色：標題色、價格色、日期色、標語底色、標語字色
- 標語文字：自由輸入
- 區塊位置：預覽圖上直接拖曳

## 輸出規格

單一 banner-studio.html，含 React 18 CDN + Babel + Canvas 匯出邏輯。
部署方式：Netlify Drop / GitHub Pages / 任意 web server / 直接瀏覽器開啟。
無後端需求，所有圖片處理在瀏覽器端完成。
